
### Importing library

You can import the generated bundle to use the whole library generated by this starter:

```typescript
import { 
  scaleSize, 
  scaleHeight, 
  setSpText2, 
  isIphoneX, 
  ifIphoneX 
} from '@apprush/react-native-responsive-design';
```
